Worko is a revolutionary platform to help Japanese Language Learners master kana and kanji. There are 3 character sets in Japanese; kanji, hiragana, and katakana. Wordoko familiarizes you with all 3 via a simple puzzle game. Using a word search design Workoko helps learners of all skill levels practice and learn hiragana, kanji and katakana.

Learn Kanji Easily

Kanji can be difficult to learn even for native speakers. Wordoko helps learners who have an understanding of hiragana and katakana can easily learn kanji. The game is presented as a word search with the word bank presented in one character set and the search area presented in another.
